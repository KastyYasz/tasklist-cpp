# TaskList C++

A simple C++ task list application for managing, editing, and completing daily tasks via the terminal.

## Features

- ✅ Add new tasks  
- 🗑️ Remove tasks  
- 📋 View all tasks  
- ✏️ Edit tasks  
- 🎉 Mark tasks as completed  
- 🧹 Clear all tasks  
- 🚪 Exit the application
